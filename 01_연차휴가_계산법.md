# HR 신입이 첫 달에 꼭 알아야 할 연차 계산법

> **카테고리:** 연차·휴가 | **난이도:** 초급 | **관련 법령:** 근로기준법 제60조

---

## 🎯 이 글을 읽고 나면 알 수 있는 것

- 입사 첫 해 연차가 몇 개인지 계산하는 방법
- 1년 이상 근속자의 연차가 몇 개인지 계산하는 방법
- 연차를 잘못 계산했을 때 발생하는 법적 리스크

---

## 📌 먼저 이것부터: 연차는 두 가지로 나뉩니다

연차를 처음 다루는 분들이 가장 많이 헷갈리는 부분이 바로 이것입니다.
**"입사 1년 미만"과 "1년 이상"은 계산 방식이 완전히 다릅니다.**

| 구분 | 대상 | 발생 방식 |
|------|------|-----------|
| 월차형 연차 | 입사 후 1년 미만 | 매달 개근 시 1일씩 발생 |
| 연간 연차 | 1년 이상 근속 | 연 15일 기본 + 근속 가산 |

---

## 📗 Case 1. 입사 1년 미만 직원의 연차

### 기본 원리

입사 후 **1개월을 개근**하면 **1일의 유급휴가**가 발생합니다.
최대 **11개월 × 1일 = 11일**이 발생 가능합니다.

> **근거 법령:** 근로기준법 제60조 제2항
> "1년 미만인 근로자 또는 1년간 80% 미만 출근한 근로자에게 1개월 개근 시 1일의 유급휴가를 준다."

### 계산 예시

```
입사일: 2025년 3월 1일
기준일: 2025년 9월 1일 (6개월 경과)

→ 연차 발생 = 6일 (3월~8월, 각 월 개근 시)
```

### ⚠️ 주의사항

- **"개근"의 기준**: 해당 월에 1일이라도 결근하면 그달 연차는 발생하지 않습니다
- **연차 사용 차감**: 1년 미만 연차를 사용한 경우, 1년이 지나 발생하는 15일에서 차감됩니다 (근로기준법 제60조 제5항)
- **퇴직 시 미사용 연차**: 퇴직 시 남은 연차는 연차수당으로 지급해야 합니다

---

## 📘 Case 2. 1년 이상 근속자의 연차

### 기본 원리

1년 이상 근속하고, **해당 연도에 80% 이상 출근**한 근로자에게 **15일의 연차**가 부여됩니다.

> **근거 법령:** 근로기준법 제60조 제1항

### 근속 연수별 가산 연차

3년 이상 근속한 경우, 2년마다 1일이 추가됩니다. 최대 25일까지 가산됩니다.

```
가산 연차 = INT((근속연수 - 1) / 2)
총 연차   = 15일 + 가산 연차  (최대 25일)
```

| 근속연수 | 기본 | 가산 | 총 연차 |
|--------|------|------|--------|
| 1년 | 15일 | 0일 | **15일** |
| 3년 | 15일 | 1일 | **16일** |
| 5년 | 15일 | 2일 | **17일** |
| 10년 | 15일 | 4일 | **19일** |
| 21년 이상 | 15일 | 10일 | **25일** (최대) |

### 계산 예시

```
입사일: 2020년 1월 1일
기준일: 2026년 1월 1일 (6년 근속)

가산 연차 = INT((6 - 1) / 2) = 2일
총 연차   = 15 + 2 = 17일
```

### ⚠️ 주의사항

- **80% 미만 출근 시**: 연간 연차 15일이 아닌 1개월 개근 당 1일만 부여됩니다
- **출근율 계산에 포함되는 것**: 연차·병가·출산휴가·육아휴직 등 법정 휴가 기간은 출근으로 간주합니다
- **연차 소멸 시효**: 미사용 연차는 발생일로부터 **1년** 이 지나면 소멸됩니다 (연차수당 청구권은 3년)

---

## 🧮 빠른 계산 공식 요약

```
📌 1년 미만
  월별 연차 = 개근한 달 수 × 1일

📌 1년 이상
  기본 연차 = 15일
  가산 연차 = INT((근속연수 - 1) / 2)  ← 최대 10일
  총 연차   = 기본 + 가산 (최대 25일)

📌 출근율
  출근율(%) = (실제 출근일수 / 소정 근로일수) × 100
```

---

## 🚨 신입 HR이 자주 하는 실수 TOP 3

1. **입사 1년 미만 월차를 별도 관리하지 않음**
   → 엑셀로 월별 개근 여부를 반드시 기록하세요

2. **1년 미만 연차 사용분을 15일에서 미차감**
   → 1년이 되는 시점에 차감 처리가 누락되지 않도록 확인

3. **육아휴직 기간을 출근율 계산에서 제외**
   → 육아휴직은 출근으로 간주해야 합니다 (근로기준법 제60조 제6항)

---

## 📋 관련 법령 요약

| 조항 | 내용 |
|------|------|
| 근로기준법 제60조 제1항 | 1년간 80% 이상 출근 시 15일 유급휴가 |
| 근로기준법 제60조 제2항 | 1년 미만 근로자: 1개월 개근 시 1일 |
| 근로기준법 제60조 제4항 | 3년 이상 근속 시 2년마다 1일 가산, 최대 25일 |
| 근로기준법 제60조 제5항 | 1년 미만 사용 연차는 1년차 연차에서 차감 |
| 근로기준법 제61조 | 사용자의 연차사용 촉진 의무 |

---

## ✅ 신입 HR을 위한 체크리스트

- [ ] 입사일 기준으로 연차 발생일을 캘린더에 표시해두기
- [ ] 1년 미만 직원은 월별 개근 여부 기록 유지
- [ ] 연차 사용 시 즉시 시스템 또는 대장에 기록
- [ ] 연도 말 미사용 연차 현황 파악 및 촉진 안내
- [ ] 퇴직 예정자의 미사용 연차 수당 정산 확인

---

> 💡 **TIP**: 연차 계산이 복잡하게 느껴진다면, 고용노동부 공식 사이트에서 제공하는 **연차휴가 계산기**를 활용해보세요. 단, 최종 판단은 반드시 근로기준법 조문을 함께 확인하세요.
