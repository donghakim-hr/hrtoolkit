# 근로계약서 작성 시 반드시 넣어야 할 필수 항목 10가지

> **카테고리:** 채용·계약 | **난이도:** 초급 | **관련 법령:** 근로기준법 제17조

---

## 🎯 이 글을 읽고 나면 알 수 있는 것

- 근로계약서에 법적으로 반드시 포함해야 할 항목
- 서면 계약서 미작성 시 처벌 수위
- 유형별(정규직/계약직/단시간) 추가 주의사항
- 실제 작성 시 자주 빠지는 항목 체크리스트

---

## 📌 근로계약서란?

근로계약서는 사용자와 근로자 사이의 **근로조건을 명시한 법적 문서**입니다.
구두 계약은 효력은 있으나, **서면 작성은 법적 의무**입니다. 작성하지 않으면 처벌받습니다.

> **근거 법령:** 근로기준법 제17조
> "사용자는 근로계약을 체결할 때 근로자에게 임금, 소정근로시간, 휴일, 연차유급휴가 등 대통령령으로 정하는 근로조건을 명시하여야 한다."

---

## 📋 법정 필수 기재 항목 10가지

### ① 임금 (급여)

단순히 금액만 쓰면 안 됩니다. 구성 항목까지 명시해야 합니다.

```
✅ 올바른 예시
  - 기본급: 월 2,500,000원
  - 식대: 월 100,000원
  - 합계: 2,600,000원
  - 지급일: 매월 25일
  - 지급 방법: 계좌이체

❌ 잘못된 예시
  - 급여: 월 2,600,000원 (항목 구분 없음)
```

> **관련 조항:** 근로기준법 제17조 제1항 제1호

---

### ② 소정근로시간

하루, 주간 단위로 명시해야 합니다.

```
✅ 올바른 예시
  - 소정근로시간: 1일 8시간, 주 40시간
  - 근무 시간: 09:00 ~ 18:00 (휴게 1시간 포함)
```

> **관련 조항:** 근로기준법 제17조 제1항 제2호

---

### ③ 휴일

주휴일과 공휴일을 명확히 기재합니다.

```
✅ 올바른 예시
  - 주휴일: 매주 일요일
  - 관공서 공휴일에 관한 규정에 따른 공휴일
```

> 💡 **2022년부터** 5인 이상 사업장은 관공서 공휴일을 **유급 휴일**로 보장해야 합니다.

> **관련 조항:** 근로기준법 제55조, 제17조 제1항 제3호

---

### ④ 연차유급휴가

연차 일수와 사용 방식을 기재합니다.

```
✅ 올바른 예시
  - 입사 후 1년 미만: 1개월 개근 시 1일 발생 (최대 11일)
  - 1년 이상: 근로기준법 제60조에 따라 부여
```

> **관련 조항:** 근로기준법 제60조, 제17조 제1항 제4호

---

### ⑤ 취업 장소

근무할 장소를 명시합니다.

```
✅ 올바른 예시
  - 취업 장소: 서울특별시 강남구 OO로 OO (본사)
  - 단, 업무상 필요 시 변경될 수 있음
```

> ⚠️ 명시된 장소 외 근무를 요청할 경우, 근로자 동의가 필요합니다.

---

### ⑥ 업무 내용 (직종)

담당 업무를 구체적으로 기재합니다.

```
✅ 올바른 예시
  - 직종: 인사총무
  - 담당 업무: 채용, 급여 관리, 4대보험 처리 등
```

---

### ⑦ 근로계약 기간

정규직이라면 "기간의 정함이 없음"으로 명시합니다.

| 고용 유형 | 기재 방법 |
|-----------|-----------|
| 정규직 | "기간의 정함이 없는 근로계약" |
| 계약직 | "2026년 4월 1일 ~ 2027년 3월 31일 (1년)" |

> ⚠️ **계약직 주의**: 계약기간이 2년을 초과하면 **무기계약직 전환 의무**가 발생합니다. (기간제 및 단시간근로자 보호 등에 관한 법률 제4조)

---

### ⑧ 사회보험 가입 여부

4대보험 가입 여부를 명시하거나, 취업규칙을 따른다고 기재합니다.

```
✅ 예시
  - 4대보험(국민연금, 건강보험, 고용보험, 산재보험) 가입
```

> **관련 조항:** 고용보험법, 국민연금법, 국민건강보험법, 산업재해보상보험법

---

### ⑨ 퇴직급여 관련 사항

퇴직연금 제도가 있다면 해당 내용을 기재합니다.

```
✅ 예시
  - 퇴직급여: 근로자퇴직급여 보장법에 따라 지급
  - 퇴직연금제도 유형: DC형 (확정기여형)
```

---

### ⑩ 서명 및 교부

근로계약서는 **2부 작성 후 각 1부씩 보관**해야 합니다.

```
✅ 올바른 절차
  1. 근로자와 사용자 모두 서명 또는 날인
  2. 사본 1부를 근로자에게 즉시 교부
  3. 원본 1부를 회사에 보관 (3년간 보존 의무)
```

> **관련 조항:** 근로기준법 제42조 (서류 보존 의무 3년)

---

## 🔍 고용 유형별 추가 체크사항

### 계약직 (기간제)

- [ ] 계약 기간 명시 (최대 2년, 초과 시 무기계약직 전환)
- [ ] 계약 갱신 조건 또는 자동 만료 조건 기재
- [ ] 갱신 기대권이 발생할 수 있는 표현 주의 ("갱신 가능성 있음" 등)

### 단시간 근로자 (파트타임)

- [ ] 시간급 또는 일급 명시
- [ ] **초과근무 동의 조항** 별도 기재 필수 (미기재 시 초과 근무 요청 불가)
- [ ] 주휴수당 지급 여부 (주 15시간 이상 근무 시 발생)

### 수습 기간이 있는 경우

- [ ] 수습 기간 명시 (통상 3개월)
- [ ] 수습 기간 중 임금 감액 여부 (최저임금의 90%까지 가능)
- [ ] 수습 종료 후 본 채용 조건 기재

## ✅ 신입 HR을 위한 최종 체크리스트

**계약서 작성 전**
- [ ] 고용 유형 확인 (정규직 / 계약직 / 단시간)
- [ ] 임금 항목 구체적으로 확정 (기본급 + 각종 수당)
- [ ] 근무 시간, 장소, 업무 내용 확정

**계약서 작성 시**
- [ ] 10가지 필수 항목 모두 포함 여부 확인
- [ ] 임금 구성 항목 세분화 기재
- [ ] 2부 작성 및 쌍방 서명

**계약서 작성 후**
- [ ] 근로자에게 1부 즉시 교부
- [ ] 사본을 3년간 보관할 폴더에 보관
- [ ] 4대보험 취득 신고 (취득일로부터 14일 이내)
