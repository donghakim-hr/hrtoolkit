# 2026년 최저임금 10,320원 — 우리 회사 급여, 위반 아닌지 지금 확인하세요

> **카테고리:** 급여·임금 | **난이도:** 초급 | **관련 법령:** 최저임금법 제6조

---

- 2026년 최저임금 기준
- 월급제·시급제별 최저임금 위반 여부 확인법
- 최저임금에 산입되는 임금과 제외되는 임금
- 위반 시 처벌 수위

## 📌 2026년 최저임금 한눈에 보기

| 구분 | 금액 |
|------|------|
| **시간급** | **10,320원** |
| **일급 (8시간 기준)** | 82,560원 |
| **월급 (209시간 기준)** | **2,156,880원** |

> **근거 법령:** 최저임금법 제8조, 2026년 최저임금 고시 (2025년 8월 고시)

> 💡 **209시간이란?**
> 주 40시간 근무 + 주휴시간(8시간) = 주 48시간
> 월 환산 = 48시간 × (365일 / 7일 / 12개월) ≈ 209시간

---

## 📗 Step 1. 우리 직원의 시급 환산하기

최저임금 위반 여부는 **시급**으로 환산해서 비교해야 합니다.

### 월급제 직원의 시급 환산

```
환산 시급 = 월 임금 총액 ÷ 월 소정근로시간

월 소정근로시간 계산 예시 (주 40시간, 주휴 포함):
  주 48시간 × (365 ÷ 7 ÷ 12) = 약 209시간
```

### 계산 예시

```
월급: 2,200,000원
근무 형태: 주 40시간, 월 209시간

환산 시급 = 2,200,000 ÷ 209 = 10,526원
최저임금 = 10,320원

→ 10,526원 > 10,320원 ✅ 위반 아님
```

---

## 📘 Step 2. 최저임금 산입 여부 확인하기

모든 임금이 최저임금 비교 대상이 되지는 않습니다.

### 최저임금 산입 항목 ✅

- 기본급
- 직무수당, 직책수당
- **매월 1회 이상 정기적으로 지급**되는 수당

### 최저임금 제외 항목 ❌

| 항목 | 이유 |
|------|------|
| 연장·야간·휴일 근로수당 | 소정근로 외 근무에 대한 대가 |
| 연간 상여금 중 월 환산액의 **15%** 초과분 | 정기성 요건 충족 시 일부 산입 |
| 식대, 교통비 등 복리후생비 중 **월 환산액의 3%** 초과분 | 실비 변상적 성격 |
| 가족수당, 근속수당 (비정기) | 정기성·일률성 미충족 |

> ⚠️ **2024년부터 변경된 사항**: 정기상여금과 복리후생비의 산입 기준 비율이 단계적으로 확대되었습니다. 연도별 기준을 반드시 확인하세요.

### 산입 범위를 적용한 비교 예시

```
기본급: 1,800,000원
식대:     200,000원 (매월 지급)
교통비:   100,000원 (매월 지급)
연장수당: 300,000원

월 환산 총액 (세전): 2,400,000원
소정근로시간: 209시간

식대 산입 한도: 2,156,880 × 3% = 64,706원
→ 식대 중 64,706원만 산입 가능 (200,000 - 64,706 = 135,294원 제외)

교통비 산입 한도: 식대와 합산하여 3% 한도 적용
→ 이미 한도 초과로 교통비 전액 제외

연장수당: 제외

최저임금 비교 대상 임금 = 1,800,000 + 64,706 = 1,864,706원
환산 시급 = 1,864,706 ÷ 209 = 8,921원

→ 8,921원 < 10,320원 ❌ 최저임금 위반!
```

## 📎 수습 기간 중 최저임금 감액

수습 중인 직원은 최저임금의 **90%** 까지 감액 지급이 허용됩니다.

| 조건 | 기준 |
|------|------|
| 대상 | 수습 기간 중인 근로자 |
| 적용 기간 | 수습 시작 후 **3개월 이내** |
| 감액 한도 | 최저임금의 **90%** |
| 예외 | 단순 노무 직종은 감액 불가 (한국표준직업분류 참고) |

```
2026년 수습 기간 최저 시급 = 10,320원 × 90% = 9,288원
수습 기간 최저 월급 = 2,156,880원 × 90% = 1,941,192원
```
